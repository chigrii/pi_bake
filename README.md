# pi_bake
raspi config files

# How to

## install some tools

The following packages will be installed,

- ssh
- byobu
- vim
- vim-gtk
- netatalk
- tightvncserver

```bash
cd ~/{YOUR_WORKSPACE}/pi_bake
./install_tools.sh
```

## setup bashrc

```bash
ln -sf ~/{YOUR_WORKSPACE}/pi_bake/.bashrc ~/.bashrc
```
